# Tetris

A fully featured Tetris clone built with C# and WPF. Standard Tetris gameplay with hold piece, next piece preview, and ghost block.

## Tech Stack
- C# (.NET)
- WPF (Windows Presentation Foundation)
- XAML

## How to Run
Open `Tetris.sln` in Visual Studio and run the project (F5).

## Features
- 7 standard Tetrimino block types with colored tile assets
- Left/Right arrow keys to move, Z/X to rotate, Space to hard drop
- Hold piece system (C key) and next piece preview
- Ghost block showing where the piece will land
- Row clearing with score tracking
- Game speed increases as score rises
- Game over screen with final score and play again option
